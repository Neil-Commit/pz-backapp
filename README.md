# Project Zomboid Backup Tool

A simple, reliable backup tool for Project Zomboid saves, designed for speed and peace of mind.
Overview

## Overview

This app was built to solve a personal problem: backing up saves while playing on the unstable version of Project Zomboid. It’s intentionally simple and focused — no unnecessary features, just quick and reliable backups.

## Features

* Backup your current save
* Add notes to each backup (so you know where you left off)
* Open the backup folder directly from the app
* Restore any previously saved backup

## Warning

Make sure you are **NOT currently playing** your save when using this program.
For best results, only run the app when the game is completely closed.

## Status

* Version: Tested and working on **Project Zomboid 42.16.1 (unstable)**.
* Reliability: Actively used in personal daily gameplay for over 4 months.

## Disclaimer

This is a personal utility tool provided "AS IS". While designed to assist with save management, the developer (NealNeil) is not responsible for any data loss. Always double-check your backups!
